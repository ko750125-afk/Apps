# 🚀 RapidForge Admin Guide

본 문서는 RapidForge 서비스의 프로젝트를 관리하는 방법을 설명합니다.

---

## 1. 🔐 로그인 (Login)
*   **경로**: `/admin/login`
*   **방법**: 사전에 등록된 관리자 계정으로 로그인합니다. (Firebase Auth 기반)
*   **보안**: 세션이 만료되면 자동으로 로그아웃됩니다.

## 2. 📊 대시보드 (Dashboard)
*   **프로젝트 조회**: 현재 등록된 모든 프로젝트를 한눈에 확인할 수 있습니다.
*   **검색**: 프로젝트명, 카테고리, 설명을 키워드로 검색할 수 있습니다.
*   **삭제**: 각 행의 삭제 버튼을 클릭하여 프로젝트를 제거할 수 있습니다. (재확인 팝업 노출)

## 3. ➕ 프로젝트 등록/수정 (Add & Edit)
*   **기본 정보**: 프로젝트 이름, 배포 URL, GitHub 저장소 경로를 입력합니다.
*   **카테고리**: 비즈니스, 예약, 어드민, 이커머스, 자동화 도구 중 선택합니다.
*   **설명 (Description)**: 카드 섹션에 노출될 짧은 요약 문구입니다.
*   **이미지 (Image)**: 프로젝트 스크린샷의 URL을 입력합니다. (추후 업로드 기능 확장 예정)
*   **상세 메모 (Memo - Markdown)**: 
    *   상세 페이지에 표시될 내용을 **Markdown** 형식으로 작성합니다.
    *   에디터에서 실시간 미리보기를 지원합니다.
    *   기술 스택, 주요 기능, 개발 후기 등을 자유롭게 기록하세요.

## 4. ⚡ 캐시 갱신 (Revalidation)
*   본 서비스는 성능 최적화를 위해 ISR(Incremental Static Regeneration)을 사용합니다.
*   관리자 페이지에서 데이터를 변경하면 **즉시 캐시 갱신(Revalidation)**이 트리거되어 홈페이지에 반영됩니다.

## 5. ⚠️ 위험 구역 (Danger Zone)
*   **경로**: `/admin/danger`
*   **기능**: 모든 프로젝트 데이터를 일괄 삭제합니다.
*   **주의**: 실행 시 복구가 절대 불가능하므로, 모든 데이터를 초기화하고 새로 시작할 때만 사용하세요.

---

## 💡 팁 (Tips)
*   상세 메모 작성 시 `### 제목` 또는 `- 리스트` 형식을 활용하면 상세 페이지가 더욱 풍성해집니다.
*   이미지는 `https://s0.wp.com/mshots/v1/[URL]?w=800` 서비스를 이용하면 웹사이트 스크린샷을 자동으로 가져올 수 있습니다.
